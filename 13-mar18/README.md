# ☕ Cafe Ordering System (Class 13)

This project demonstrates **Object-Oriented Programming (OOP)** in Python by building a functional command-line coffee shop system. It features an interactive menu using Python's latest **match/case** logic (instead of traditional if/elif blocks) and automatic receipt generation.

---

## ✅ Assignment Checklist

1. [x] **Step 1:** Create the `Coffee` Class
2. [x] **Step 2:** Create the `Order` Class
3. [x] **Step 3:** Create the `Cafe` Class
4. [x] **Step 4:** Set Up the Menu Data
5. [x] **Step 5:** Build the Interactive Menu (The `while` Loop with **match/case** logic)
